# AI Usage Guide

Use Claude to:
- Summarize clients
- Recall past decisions
- Draft project plans
- Rewrite product descriptions
- Search development knowledge
- Reuse prompts

Best prompts:
- "Summarize everything we know about Client X"
- "List all active projects and risks"
- "Use my standard code review prompt"
- "Find similar past implementations"

Rules:
- Always update overview notes
- Let AI read first, write second
- Store final results, not raw output
